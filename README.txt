망고 스쿠버 홈페이지 수정본

사용 방법
1. 이 폴더를 VS Code로 열어주세요.
2. index.html 파일을 Live Server로 실행하세요.

포함된 파일
- index.html
- style.css
- images/logo.png

건물사진 넣는 방법
1. images 폴더 안에 건물 사진을 넣어주세요.
2. 파일명을 building.jpg 로 바꿔주세요.
3. index.html을 새로고침하면 메인 오른쪽 카드에 사진이 표시됩니다.

수정 내용
- 메뉴 3개: 가격안내 / 시설소개 / 오시는길
- 메뉴 가운데 정렬
- 오른쪽 카드 안 로고 제거
- 건물 사진을 위로 이동
- 가격안내, 시설소개, 오시는길 섹션 추가


추가 수정 내용
- 카카오톡 오픈채팅 링크 추가
- 메인 문의 버튼에 카카오톡 문의 추가
- 오시는길 정보와 버튼에 카카오톡 문의 추가
- 화면 오른쪽 아래 카카오톡 고정 문의 버튼 추가
